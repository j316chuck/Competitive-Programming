                  int dist[][] = new int[V][V];
                  int i, j, k;
                  for (i = 0; i < V; i++)
                  for (j = 0; j < V; j++)
                  dist[i][j] = graph[i][j];
                  for (k = 0; k < V; k++){
                  for (i = 1; i < V; i++){
                  for (j = 0; j < V; j++){
                  if (dist[i][k] + dist[k][j] < dist[i][j])
                  dist[i][j] = dist[i][k] + dist[k][j];

                                                                                                      see_language Saurabh a year ago

                                                                                                      The outer most loop picks a vertex(k) to check if it is in the shortest path from the vertex i (second loop) to vertex j (inner most loop). Initially the matrix contains the paths of length one(length means number of edges). On every iteration of the outer most loop we can say that a value M[i][j] is updated if there is a path of length k which is smaller then the previous smallest path.
